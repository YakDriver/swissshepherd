A documentation linter for Terraform providers. swissshepherd compares a provider's schema against its Markdown documentation and reports missing, phantom, misordered, mislabeled, or misformatted docs — covering everything from Read-Only attribute coverage and section presence to frontmatter, byline, and heading style.

📖 **Full documentation:** <https://yakdriver.github.io/swissshepherd/>

```bash
go install github.com/YakDriver/swissshepherd@latest
swissshepherd
```
